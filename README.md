# booking-poc
PoC for a real-time booking app using WebSockets.

## Install
```
git clone https://github.com/buchslava/booking-poc.git
cd booking-poc
npm i
```

## Run server
```
npm start
```

## Run admin
```
npm run admin
```

### Add tickets
1. Run admin
2. Enter tickets quantity as a integer
3. Hit Enter

### Print ticket holders
1. Run admin
2. Enter `!`
3. Hit enter
4. Switch to the server window
